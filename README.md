# CPay-Web
免签四方支付项目

#QrPay是我接手时期的项目了

这是远古版本的qrpay了 当时是外包的技术写的 很烂

叫CPay是本人想的一种象征含义（ChinaPersonalPay 正好两个P连一块就删掉一个嘻嘻）

这个系统，支持微信、支付宝、QQ钱包的支付方式，无需官方接口对接，个人即可。    
全部免挂机，对接方式：易支付，可以拿来运营
